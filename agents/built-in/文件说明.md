# agents/built-in/  — 内置 Agent 定义

YAML 格式的 AgentDefinition 模板。启动时由 `AgentRegistry.LoadFromDir()` 自动加载。

## 文件列表

| 文件 | agent_type | 工具策略 | 超时 | 上下文 | 记忆 |
|------|-----------|---------|------|--------|------|
| `general.yaml` | general | 全部(减orchestration) | 5m | task | none |
| `researcher.yaml` | researcher | 全部(减orchestration) | 10m | summary | persistent |
| `code-reviewer.yaml` | code-reviewer | 只读工具 | 5m | summary | persistent |

## 使用方式

LLM 调用 `spawn_agent(agent_type="researcher")` 时，AgentRegistry 解析对应 YAML，
将 `DisallowedTools` / `MaxTimeout` / `SystemPrompt` 注入子 agent 的工厂和 prompt。
