# persistence 包 — 文件说明

SQLite + GORM 持久化层。所有表由 `session_id` (= `conversations.ID`) 关联。

## 表结构 (6 张)

```
conversations (元数据)
  ↓ session_id
  ├── history_messages    — 对话消息 (Agent Loop)
  ├── worker_transcripts  — Subagent 执行记录
  ├── session_token_usages — Token 成本统计
  └── evo_traces → evo_evaluations → evo_repairs (进化链)
```

## 文件

| 文件 | 职责 |
|------|------|
| `db.go` | SQLite 连接 + `Conversation` 模型 + WAL 模式 |
| `repository.go` | Conversation CRUD + 级联删除（删 session 清全部 6 表） |
| `history.go` | `HistoryMessage` — Agent Loop 对话持久化 |
| `transcript.go` | `WorkerTranscript` — Subagent 工具调用追踪 |
| `token_usage.go` | `SessionTokenUsage` — 每 session 的 token 成本 |
| `evo.go` | `EvoTrace/Evaluation/Repair` — 进化自评估链 |
