# 文件说明

## 项目概述

本项目为 **NGOAgent** — 一个生产级 AI Agent 框架，采用 DDD 架构（Domain-Driven Design），实现了完整的 ReAct 循环、安全沙箱、多模态支持和知识提炼系统。

## 架构概览

```
cmd/                          # 入口：agent CLI
internal/
  application/                # DI 总线（builder.go）
  domain/
    service/                  # 核心 Agent Loop（ReAct + 状态机）
    tool/                     # 工具接口 + 元数据
  infrastructure/
    llm/                      # LLM Provider Router + 遥测
    tool/                     # 30+ 工具实现
    security/                 # 安全沙箱（pattern/llm/hybrid 分类器）
    notify/                   # Webhook 通知
    prompt/                   # PromptEngine + 附件引擎
    skill/                    # Skill 管理（KI 分类）
    workspace/                # 工作区上下文（@include 递归）
  interfaces/
    server/                   # HTTP + WebSocket API
    grpc/                     # gRPC API
webui/                        # React 前端
```
