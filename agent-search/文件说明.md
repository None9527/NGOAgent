# Agent Search Service

从 SearXNG + TEI Reranker + Crawl4AI + devlabz-search (4 容器) 合并为单一 Python 服务。

## 架构

```
NGOAgent → Agent Search Service (本服务) → SearXNG (localhost:8080)
               │                                
               ├─ Stage 1: 算法预筛 (URL 去重/域名多样性/引擎共识/时效性)
               ├─ Stage 2: LLM 重排 (DashScope qwen3.5-27b, 4 维度评分)
               └─ agent-browser 并发深度抓取 (Playwright)
```

## 文件结构

```
agent-search/
├── main.py              # FastAPI 入口 (3 个 API 端点)
├── config.py            # 环境变量配置
├── requirements.txt     # 依赖
├── search/
│   ├── __init__.py
│   ├── models.py        # Pydantic 数据模型
│   ├── searxng.py       # SearXNG 客户端 (完整字段解析)
│   ├── prefilter.py     # Stage 1 算法预筛
│   ├── reranker.py      # Stage 2 LLM 精排
│   └── fetcher.py       # agent-browser 并发页面抓取
└── 文件说明.md           # 本文件
```

## API 端点

| 端点 | 用途 |
|------|------|
| `POST /api/search` | 搜索 + 预筛 + LLM 重排 (不抓取页面) |
| `POST /api/fetch` | 深度抓取单个 URL |
| `POST /api/search_and_fetch` | 搜索 + 重排 + 自动深度抓取 Top N |

## 启动

```bash
export LLM_API_KEY="sk-xxx"
cd agent-search
pip install -r requirements.txt
uvicorn main:app --host 0.0.0.0 --port 8889
```

## 环境变量

| 变量 | 默认值 |
|------|--------|
| `SEARXNG_URL` | `http://localhost:8080` |
| `LLM_URL` | `https://dashscope.aliyuncs.com/compatible-mode/v1` |
| `LLM_API_KEY` | (必填) |
| `LLM_MODEL` | `qwen3.5-27b` |
