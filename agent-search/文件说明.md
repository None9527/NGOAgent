# Agent-Search Service — 文件说明

## 架构

统一的 AI 搜索管道服务，为 NGOAgent 提供 web 情报能力。

```
POST /api/search   — 统一搜索 (depth: quick/standard/deep)
POST /api/extract  — 已知 URL 内容提取
```

管道: `SearXNG → 算法预筛 → LLM 重排 → 并发抓取 → 算法去噪`

## 文件结构

| 文件 | 用途 |
|---|---|
| `main.py` | FastAPI 入口，2 个端点 |
| `config.py` | 环境变量配置 + 域名质量列表 |
| `search/models.py` | 数据模型 (API + 内部管道) |
| `search/pipeline.py` | 管道编排器 — 根据 depth 调度各 Stage |
| `search/searxng.py` | Stage 0: SearXNG 客户端 |
| `search/prefilter.py` | Stage 1: 算法预筛 (去重 + 域名多样 + 评分) |
| `search/reranker.py` | Stage 2: LLM 重排 (score + reason + result_type + intent) |
| `search/fetcher.py` | Stage 3: L1/L2 反爬抓取 + `force_stealth` 主动潜行模式 + 二进制缓存 |
| `search/cleaner.py` | Stage 4: 算法去噪 (导航/广告/页脚/评论清除) |

## 运行

```bash
cd agent-search
uvicorn main:app --host 0.0.0.0 --port 8889
```

环境变量:
- `SEARXNG_URL` — SearXNG 实例地址 (默认 `http://localhost:8080`)
- `LLM_URL` — LLM API 地址
- `LLM_API_KEY` — LLM API 密钥
- `LLM_MODEL` — 重排模型 (默认 `qwen-turbo`)
- `CACHE_DIR` — 二进制缓存目录 (默认 `data/cache`)
