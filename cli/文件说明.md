# CLI (Node.js TUI)

基于 Ink + React + TypeScript 的终端 TUI，全面对齐 Claude Code 架构。
通过 HTTP/SSE 与 Go 后端 AgentAPI 通信。

## 目录结构

```
cli/
├── src/
│   ├── index.tsx              # 入口 (--server 参数，默认 http://localhost:8080)
│   ├── api/
│   │   └── client.ts          # HTTP/SSE 客户端 (26 个 API 方法)
│   ├── grpc/
│   │   └── client.ts          # [旧] gRPC 客户端 (已弃用，保留备份)
│   ├── components/
│   │   ├── App.tsx            # 6 态状态机 (idle/streaming/selecting/approving/help/history/diffing)
│   │   ├── Banner.tsx         # 启动横幅
│   │   ├── StatusBar.tsx      # CC 风格状态栏 (model│ctx%│cost│mode)
│   │   ├── MessageList.tsx    # 消息渲染 (ToolCallBlock/ThinkingBlock 折叠)
│   │   ├── InputArea.tsx      # 输入区 (Tab 补全 + 多行 + 历史)
│   │   ├── PermissionRequest.tsx  # 权限审批 (y/a/n/d + Ctrl+E 详情)
│   │   ├── Selector.tsx       # 通用选择器 (↑↓/Enter/Esc)
│   │   ├── HelpDialog.tsx     # CC 风格帮助弹窗 (Esc 关闭)
│   │   └── Autocomplete.tsx   # 命令补全弹出列表 (↑↓/Tab)
│   └── commands/
│       └── index.ts           # 23 个 slash 命令 + Tab 补全
├── package.json
├── tsconfig.json
└── dist/                      # tsup 打包输出
```

## 技术栈

- **运行时**: Node.js v22+
- **UI**: Ink 5 + React 18
- **通信**: HTTP fetch + SSE (Server-Sent Events)
- **构建**: tsup (ESM, ~68KB)
- **开发**: tsx (热重载)

## 命令

```bash
npm install          # 安装依赖
npm run dev          # 开发模式
npm run build        # 生产构建
npm start            # 运行
```

## 后端 API 架构

```
CLI (HTTP/SSE) → Server (HTTP 适配器) → AgentAPI (统一 facade) → 核心服务
```

所有 26 个 API 方法通过 `AgentAPI` 统一调度，协议无关。
未来加 gRPC 只需写薄适配器，核心代码零改动。
